# 🌸 PeriodSathi Nepal (पिरियड साथी)

> "तपाईंको स्वास्थ्य, तपाईंको शक्ति" — Your Health, Your Strength

A menstrual health awareness and tracking platform designed specifically for Nepali women, addressing cultural stigma, misinformation, and unequal healthcare access.

## 🎯 Problem Statement

88% of Nepali women use unsafe menstrual materials. 71% of girls miss school during their period. Harmful practices like Chhaupadi still affect women in far-western Nepal. Menstrual stigma prevents open conversation about health.

## 💡 Our Solution

A bilingual English/Nepali web platform that provides:

- Simulated AI chatbot for anonymous health questions
- Period tracker with localStorage persistence
- Day-by-day menstrual wellness guide
- Myths vs Facts section combating misinformation
- Hospital and NGO finder across all 7 provinces
- Anonymous community forum simulation
- Donation and admin dashboard preview pages

## 🛠️ Tech Stack

This hackathon repo is intentionally frontend-only and dependency-free so it works directly from GitHub or `file://`.

- HTML
- CSS
- JavaScript
- LocalStorage
- Local SVG assets

The UI follows the requested PeriodSathi Nepal design direction: bilingual copy, dark/light theme, glass cards, cultural motifs, responsive mobile navigation, and offline-only hardcoded data.

## 🚀 Quick Start

Open `index.html` directly in a browser.

Optional validation:

```bash
npm run build
```
